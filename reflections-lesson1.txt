¿Como puede utilizar el commando git log y git diff para ver el historial de los archivos?

el comando git log muestra todos los commit que se hicieron en el proyecto con un Id unico para cada commit y el comando git diff muestra lo que se realizo entre 2 commit como lineas nuevas que se elimino o que se modifico.
